# 📋 Инструкция и план проекта: Random Cartoon App

Эта инструкция описывает текущее состояние проекта, результаты аудита безопасности и зависимостей, а также пошаговый план разработки.

---

## 1. Текущее состояние и стек технологий

Проект состоит из двух частей:
- **Client (Frontend):** Написан на React + Vite + Tailwind CSS (Важно: в старом README указан Angular, это неактуально).
- **Server (Backend):** Node.js + Express + Mongoose (MongoDB) + Telegraf (Telegram-бот).

---

## 2. Результаты аудита и обновления зависимостей

По результатам проверки `npm outdated` и `npm audit` выявлен ряд проблем, требующих решения на первом этапе:

### ⚙️ Server:
- **Уязвимости (Audit):** 12 уязвимостей (8 High, 4 Moderate). Затрагивают `axios`, `mongoose`, `multer`, `body-parser` и др.
- **Устаревшие пакеты:**
  - `axios` (Current: 1.11.0 → Latest: 1.18.1)
  - `express` (Current: 5.1.0 → Latest: 5.2.1)
  - `mongodb` (Current: 6.18.0 → Latest: 7.5.0)
  - `mongoose` (Current: 8.16.5 → Latest: 9.7.4)
  - А также `cors`, `dotenv`, `jsonwebtoken`, `multer`, `nodemon`.

### 💻 Client:
- **Уязвимости (Audit):** 17 уязвимостей (1 Critical, 9 High, 5 Moderate, 2 Low). Затрагивают `react-router`, `vite`, `form-data`, `rollup`, `tar` и др.

**Рекомендация:** В обеих директориях необходимо выполнить `npm audit fix`, а также обновить устаревшие пакеты для обеспечения безопасности перед началом активной разработки.
